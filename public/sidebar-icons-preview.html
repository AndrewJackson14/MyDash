Why these changes

The main problem today is icon collisions: six glyphs are shared across two or more menu items, which means users can't tell menu entries apart without reading the label. Every change below either (a) resolves a collision or (b) picks a glyph that better matches the page's actual function.

ItemBeforeAfterWhy
Editorial Hamburger-lines was generic. Newspaper layout reads as "editorial" at a glance. collision shared with Classifieds.
Newsletters Distinguishes from Mail. Paper-airplane reads as "broadcast send." collision
Knowledge Base Open book matches "read docs / learn." File icon was generic. collision shared with Email Templates.
Creative Jobs Briefcase = work/job queue. Design Studio keeps the paintbrush. collision
Classifieds Megaphone fits "small-ads / notices." collision
Merch Shopping bag is unambiguous retail. Star was decorative, not semantic.
Circulation Delivery truck matches "physical distribution." Frees publication icon for Publications only. collision
Performance Heartbeat pulse reads as "live performance monitoring." Bar chart stays on Reports. collision
Schedule Clock is the universal "when" glyph. File icon had nothing to do with schedules.
Email Templates Block-grid reads as "template/layout." Frees file icon entirely. collision
