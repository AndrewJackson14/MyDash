# Navigation overview — where things live

The left sidebar groups pages by function:

- **Home** — Dashboard (your daily priorities and issue pressure)
- **Sales** — SalesCRM (pipeline, clients, proposals, inquiries, outreach)
- **Design** — Design Studio (ad projects, briefs, proofs)
- **Editorial** — stories, writers, review queue
- **Production** — Flatplan, Issue Schedule, EZSchedule, Print Runs
- **Circulation** — subscribers, drivers, routes
- **Billing** — invoices, payments, AR aging, bills, receivables
- **Publications** — publication settings, ad sizes, goals
- **Analytics** — Reports (Overview, P&L, YoY, Revenue vs. Goals, Sales by Issue, Editorial, Subscribers, Audience)
- **Team** — team members, profiles, permissions
- **Ops** — Service Desk, Messaging, Mail, Legal Notices, Media Library
- **Integrations** — QuickBooks, Gmail, Google Calendar, Stripe, StellarPress

**Deep-linking:** clicking a client name, publication, issue, or any other named entity in a row or card jumps directly to that entity's canonical page. No retracing steps.

**Floating MyHelper button** lives in the bottom-right on every page. Press `⌘/` (Mac) or `Ctrl+/` (Windows) to open it from anywhere.
