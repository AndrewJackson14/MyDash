# Creating, sending, and converting proposals

**To create a proposal:** Go to **Sales → Proposals → + Proposal**. Fill in:

1. **Client** (pick from the dropdown or search)
2. **Proposal name** (e.g. "Q1 2026 Multi-Publication Package")
3. **Lines** — each line = one publication + issue + ad size + price. Click **+ Add Line** for each placement. Every line must be assigned to a specific issue before the proposal can be converted.
4. **Discount %** (optional) — applied to the total
5. **Payment terms** — one-time, monthly, per-issue
6. **Term months** (if monthly)

**To send a proposal:** on the proposal detail page, click **Send**. It emails the client a signature link. Status changes from Draft → Sent.

**Client signs:** they click the link, review terms, and sign electronically. Status → Signed & Converted.

**Conversion creates:**
- A Contract (with contract lines mirroring the proposal lines)
- One Sales row per proposal line (with status = Closed)
- One Ad Project per sale (for display ads)
- An invoice per issue (monthly plans) or a single lump-sum invoice (one-time plans)

**Editing after sending:** you can't edit a proposal once it's Sent. Cancel and create a new one if terms change.
